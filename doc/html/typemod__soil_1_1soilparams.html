Detailed Description

This type provides information on the soil (type of reflection model, soil properties, ...) used as input of the subroutine soil_coefficients_multiangle. The imodel and temp variables are required. Other variables are used or not depending on imodel (see the code of soil_coefficients_multiangle for details)
